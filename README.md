## はじめに

本パーツ管理システムは、製造業における部品管理を効率化し、生産性を向上させることを目的として開発されました。このREADMEでは、システムの概要、主な機能、使用方法について説明します。

## システム概要

本システムは、製造業における部品管理を階層構造で実現します。タイプ、モデル、ブロック、パーツ、バージョンという5つの階層で部品を管理し、効率的な検索や関連付けを可能にします。

## 主な機能

1. **階層構造による部品管理**
   - タイプ > モデル > ブロック > パーツ > バージョンの5階層で管理
   - 各階層での柔軟な検索と表示

2. **CRUD操作**
   - 各エンティティ（タイプ、モデル、ブロック、パーツ、バージョン）の作成、読み取り、更新、削除

3. **関連付け機能**
   - ブロックとパーツの多対多関連付け
   - モデルとブロックの関連付け

4. **検索可能なドロップダウン**
   - 大量のデータからの効率的な選択を可能にするカスタム実装

5. **バージョン管理**
   - パーツの変更履歴を追跡し、各バージョンの詳細を管理

3. **ブロック**
   - 型式を構成する機能ブロック
   - 例：ピストン、ドア、ライトユニット

4. **パーツ**
   - 実際の部品
   - 例：ピストンリング、ドアハンドル、LEDバルブ

5. **バージョン**
   - パーツの変更履歴を管理
   - 例：ピストンリングv1.0、ピストンリングv1.1

各エンティティの関連：

- タイプは複数のモデルを持つ
- モデルは複数のブロックを持つ
- ブロックは複数のパーツを持つ（多対多関連）
- パーツは複数のバージョンを持つ

## 使用方法

### 1. ログイン

システムにアクセスし、提供された認証情報でログインします。

### 2. ダッシュボード

ログイン後、ダッシュボードが表示されます。ここから各機能にアクセスできます。

### 3. 部品の登録

1. 「新規登録」ボタンをクリック
2. 階層（タイプ、モデル、ブロック、パーツ）を選択
3. 必要情報を入力
4. 「保存」をクリック

### 4. 部品の検索

1. 検索バーに検索したい部品名や型番を入力
2. 検索結果から目的の部品を選択

### 5. 部品の編集

1. 部品の詳細ページを開く
2. 「編集」ボタンをクリック
3. 情報を更新
4. 「保存」をクリック

### 6. バージョン管理

1. パーツの詳細ページを開く
2. 「新しいバージョンを作成」をクリック
3. 変更内容を入力
4. 「保存」をクリック

### 7. ブロックとパーツの関連付け

1. ブロックの詳細ページを開く
2. 「パーツを追加」をクリック
3. 検索可能なドロップダウンから関連付けるパーツを選択
4. 「保存」をクリック

### 8. モデルとブロックの関連付け

1. モデルの詳細ページを開く
2. 「ブロックを追加」をクリック
3. 検索可能なドロップダウンから関連付けるブロックを選択
4. 「保存」をクリック

## よくある質問（FAQ）

Q1: パーツの検索が遅いです。どうすればよいですか？
A1: 検索キーワードをより具体的にすることで、検索速度が向上します。また、タイプやモデルなどの上位階層から絞り込むことも効果的です。

Q2: 誤って部品を削除してしまいました。復元できますか？
A2: 現在、削除された部品の自動復元機能はありません。ただし、システム管理者にお問い合わせいただくことで、バックアップからの復元が可能な場合があります。

Q3: 新しいタイプやモデルを追加するにはどうすればよいですか？
A3: システム管理者権限が必要です。管理者メニューから「新規タイプ/モデル追加」を選択し、必要情報を入力してください。

## トラブルシューティング

1. **システムにアクセスできない**
   - インターネット接続を確認してください
   - VPNを使用している場合は、正しく接続されているか確認してください
   - 問題が解決しない場合は、担当者にお問い合わせください

2. **検索結果が表示されない**
   - スペルミスがないか確認してください
   - フィルターが正しく設定されているか確認してください
   - ブラウザのキャッシュをクリアして再試行してください

3. **変更が保存されない**
   - 必須フィールドがすべて入力されているか確認してください
   - ブラウザの更新ボタンを押して再度操作を行ってください
   - 問題が続く場合は、別のブラウザで試してみてください
